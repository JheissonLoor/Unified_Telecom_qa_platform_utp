# Reportes generados

Los subdirectorios contienen resultados locales y estan ignorados salvo sus
archivos README. Nunca deben almacenar tokens, contrasenas, SDP sin redactar o
datos personales.

- `trivy/`: vulnerabilidades, secretos y configuracion.
- `sonarqube/`: exportes o capturas del quality gate.
- `test-results/`: unitarias, integracion, E2E y carga.
- `screenshots/`: evidencias visuales redactadas.
